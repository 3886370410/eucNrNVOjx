# Python图书馆管理系统 python339

## 项目概述

本项目是基于Python语言的图书馆管理系统，集成了django框架、mysql数据库以及vue前端技术，实现了图书信息管理、借阅归还流程、用户留言等功能的综合性应用。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 系统主要功能

- 图书信息管理
- 图书馆信息展示
- 留言板
- 公告信息发布
- 个人中心
- 借阅信息管理
- 归还信息管理

### 管理功能

- 图书类型管理
- 图书管理
- 罚金信息管理
- 留言板管理

## 系统角色

- 系统管理员：负责整体系统的维护与信息管理
- 普通用户：可以进行图书借阅、归还以及留言等操作

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- Vue.js前端框架
